# Setup de Lovable

Esta sección explica cómo configurar un proyecto en Lovable para obtener mejores resultados: más consistencia, menos retrabajo y, sobre todo, más velocidad al construir.

Aquí vamos a definir la configuración base que usamos en Landscapes (estructura de documentación, knowledge, estándares y convenciones) para que Lovable:

- Entienda rápidamente el contexto del proyecto,
- Aplique guías según el tipo de tarea,
- Genere cambios pequeños y consistentes.

En los siguientes documentos entraremos en cada paso específico del setup (uno por tema) con instrucciones claras para implementarlo en cualquier proyecto.
