# ✅ Estándares de calidad

Esta sección define los lineamientos de calidad para frontend, con foco en control de cambios, observabilidad y confiabilidad funcional.

El objetivo es asegurar que cada entrega mantenga estándares técnicos consistentes y permita detectar, mitigar y prevenir riesgos en producción.

   Guía de observabilidad con Sentry para captura, monitoreo y gestión de errores.
