# 🤝 Code Standards

Esta sección establece los estándares de desarrollo frontend para mantener consistencia técnica, legibilidad y calidad en todo el código del proyecto.

El objetivo es alinear criterios de implementación que impactan directamente en:

- mantenibilidad del código,
- escalabilidad del proyecto,
- y colaboración efectiva entre equipos.

1. [Buenas prácticas](/docs/frontend/standards/good-practices)  
   Reglas y patrones para mejorar legibilidad, manejo de asincronía, composición y calidad general del código en React.
2. [Formularios](/docs/frontend/standards/forms)  
   Estándares para modelado de formularios, validaciones y separación de responsabilidades por capa.
